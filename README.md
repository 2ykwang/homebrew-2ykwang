# Homebrew Tap for mac-cleanup-go

Homebrew formulae for [mac-cleanup-go](https://github.com/2ykwang/mac-cleanup-go).

## Installation

```bash
brew tap 2ykwang/mac-cleanup-go
brew install mac-cleanup-go
```

## Update

```bash
brew update
brew upgrade mac-cleanup-go
```

## Uninstall

```bash
brew uninstall mac-cleanup-go
brew untap 2ykwang/mac-cleanup-go
```
